## ✅ 算法验证结论（重要）

经检查，`getBazi` 和 `generateNames` 云函数**已经接入 `lunar-javascript` 库**，八字计算是准确的：
- ✅ 年柱、月柱、日柱、时柱 均用节气/干支公式计算
- ✅ 农历转换使用 `Lunar` 类，精确到每一天
- ✅ 月柱使用 `lunar.getMonthInGanZhi()`，按节气分月

**结论**：算法无需修复，直接进入文案+UI重构阶段。

## 一、文案重构方案

### 核心原则
- 底层计算不变（五行八字照旧）
- 前端文案全面换皮，用诗经论语风格表达
- 审核红线：完全不提"算命""占卜""卦象"等词

### 文案替换对照表

| 现有文案/功能 | 改为 | 备注 |
|-------------|------|------|
| 八字分析 | 出生时间解读 / 生辰密码 | 不提"八字" |
| 八字 | 生辰 / 出生时间 | 正面中性 |
| 五行（金木水火土） | 五维能量 / 五种特质 | 不出现"五行" |
| 五行分布图 | 能量分布图 | 温和表达 |
| 五行缺水/木/火 | 缺少X能量 | 不说"缺" |
| 喜用神 | 性格天赋 / 优势特质 / 能量密码 | 完全替换 |
| 补五行 | 补充X能量 | 积极导向 |
| 卦象/易经 | 诗词典故 / 古文寓意 / 智慧启示 | 替换核心功能 |
| 算命/测算 | 智能推荐 / 个性化分析 | 正向表达 |
| "凶" | 一般/中等/良好 | 不用评判词 |
| "吉" | 上吉/优良/良好 | 温和表达 |
| 八字合婚（如果存在） | 名字匹配度 | 完全不提合婚 |

### 页面文案替换示例

**首页**：
- "八字分析" → "出生时间解读"
- "AI取名" → "智能起名"
- "五行平衡" → "能量匹配"

**八字确认页**：
- "八字排盘" → "生辰解读"
- "日主：戊土" → "代表：戊土属性"
- "喜用神：水、金" → "天赋特质：水、金能量"

**结果页**：
- "卦象解读" → "典故寓意"
- "《易经》·兑卦" → "《诗经》云..."
- "总格凶" → "总格偏弱"（或直接不显示凶格）

**详情页**：
- "此命旺夫" → "此名温婉得体"
- "克父" → 删除相关描述
- "事业型名字" → "进取型名字"

## 二、UI风格重构

### 风格定位
**清新插画风 + 诗意留白**
- 目标用户：新生儿父母（宝妈，25-35岁）
- 推广渠道：小红书（种草）、视频号（传播）
- 整体调性：温暖、亲切、有文化感但不老气

### 色彩规范

```css
/* 主色调 - 淡雅诗意 */
--primary: #6B8E7B;        /* 艾绿 - 主按钮、强调 */
--primary-light: #A8C5B5;  /* 淡绿 - 背景点缀 */
--primary-dark: #4A6356;   /* 深绿 - 文字 */

/* 背景色 - 温暖米调 */
--bg-primary: #FDF8F3;     /* 米白 - 主背景 */
--bg-card: #FFFFFF;         /* 纯白 - 卡片 */
--bg-accent: #F5EDE4;      /* 淡米 - 区块背景 */

/* accent色 - 朱砂点缀 */
--accent: #C67B5C;         /* 赭石 - 重点标记 */
--accent-light: #E8C4B8;   /* 淡赭 - 标签背景 */

/* 五行能量色（改叫能量色） */
--energy-gold:   #C9A96E;  /* 金 - 能量金 */
--energy-wood:   #7BA08A;   /* 木 - 能量木 */
--energy-water:  #6B8FA8;   /* 水 - 能量水 */
--energy-fire:   #C47B5C;   /* 火 - 能量火 */

/* 功能色 */

--warning: #C9A96E;
--text-primary: #3D3D3D;
--text-secondary: #7D7D7D;
--text-hint: #ADADAD;
```
