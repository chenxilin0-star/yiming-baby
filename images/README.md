# 图片资源说明

本项目需要以下图片资源，请按照规格准备：

## 必需图片

### 1. hero-bg.png
- **用途**：首页背景图
- **尺寸**：750 x 800 px
- **风格**：水墨山水、淡雅、新中式
- **格式**：PNG（带透明度更佳）

### 2. default-avatar.png
- **用途**：默认用户头像
- **尺寸**：200 x 200 px
- **风格**：简约人物剪影或头像
- **格式**：PNG

### 3. empty-fav.png
- **用途**：空收藏状态图标
- **尺寸**：200 x 200 px
- **风格**：简约的星星/收藏图标
- **格式**：PNG

### 4. share-cover.png
- **用途**：分享封面图
- **尺寸**：500 x 400 px
- **风格**：品牌展示、包含"易名宝宝"Logo
- **格式**：JPG 或 PNG

### 5. 底部标签栏图标
- **用途**：tabBar 导航图标
- **尺寸**：48 x 48 px（实际显示81x81，系统会自动处理）
- **风格**：线性图标，简洁
- **格式**：PNG
- **列表**：
  - tab_home.png - 首页未选中
  - tab_home_active.png - 首页选中
  - tab_user.png - 我的未选中
  - tab_user_active.png - 我的选中

## 图片存储位置

图片可以存储在：
1. 本地：直接放在 `images/` 目录下
2. 云存储：上传到云开发存储，使用 `cloud://` 协议引用

## 替代方案

如果没有设计资源，可以使用以下替代方案：

1. **hero-bg.png**：使用CSS渐变背景替代
2. **default-avatar.png**：使用默认的微信头像
3. **empty-fav.png**：使用emoji或系统图标
4. **share-cover.png**：使用截图
5. **tab图标**：使用emoji或文字

## 注意事项

- 图片大小尽量控制在100KB以内
- 使用压缩工具优化图片
- 云存储中的图片记得设置权限为"所有用户可读"
