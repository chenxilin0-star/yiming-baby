# AI周易取名宝宝小程序 - UI/UX交互设计文档

**文档版本**：v1.0  
**设计原则**：专业感 + 亲和力 + 文化韵味  
**视觉风格**：新中式极简，水墨意境，现代留白

---

## 1. 设计系统规范

### 1.1 色彩体系

```css
/* 主色调 - 玄青水墨 */
--primary-900: #1a1a1a;    /* 主文字 */
--primary-700: #4a4a4a;    /* 次要文字 */
--primary-500: #8a8a8a;    /* 辅助文字 */
--primary-200: #e8e8e8;    /* 分割线 */
--primary-50:  #fafafa;    /* 背景灰 */

/*  accent色 - 朱砂点缀 */
--accent-600: #c45c48;     /* 主按钮、重点标记 */
--accent-100: #fdf2f0;     /* 轻提示背景 */

/* 五行色（用于八字分析） */
--wuxing-gold:   #d4af37;  /* 金 */
--wuxing-wood:   #5a9a5a;  /* 木 */
--wuxing-water:  #4a90d9;  /* 水 */
--wuxing-fire:   #e74c3c;  /* 火 */
--wuxing-earth:  #8b7355;  /* 土 */

/* 功能色 */
--success: #52c41a;
--warning: #faad14;
--error:   #f5222d;
```

### 1.2 字体规范

| 层级 | 字号 | 字重 | 用途 |
|-----|------|------|------|
| 标题一 | 24px | Bold | 页面大标题 |
| 标题二 | 20px | Bold | 卡片标题 |
| 标题三 | 17px | Medium | 小标题 |
| 正文 | 15px | Regular | 主要内容 |
| 辅助 | 13px | Regular | 说明文字 |
| 标注 | 12px | Regular | 时间、标签 |

**字体栈**：`"PingFang SC", "Hiragino Sans GB", "Microsoft YaHei", sans-serif`

### 1.3 间距系统

```css
--space-xs: 4px;
--space-sm: 8px;
--space-md: 16px;
--space-lg: 24px;
--space-xl: 32px;
--space-xxl: 48px;

/* 圆角 */
--radius-sm: 4px;
--radius-md: 8px;
--radius-lg: 12px;
--radius-full: 9999px;
```

### 1.4 图标系统

- 使用 **阿里巴巴Iconfont** 定制图标库
- 线条风格，2px描边，24×24标准尺寸
- 关键图标：八字、五行、卦象、收藏、分享、时间

```

- 首页无返回按钮，左侧为Logo
- 透明导航：背景透明，文字白色（用于结果页顶部）
```

### 2.2 按钮规范

| 类型 | 样式 | 状态 | 用途 |
|-----|------|------|------|
| 主按钮 | 背景#c45c48，文字白色，圆角24px | 正常/禁用/加载 | 主要操作（生成、分享） |
| 次按钮 | 背景透明，边框#c45c48，文字#c45c48 | 正常/禁用 | 次要操作（修改、重填） |
| 文字按钮 | 文字#c45c48，无背景 | 正常 | 辅助操作（查看详情） |
| 图标按钮 | 48×48圆形，背景#f5f5f5 | 正常/激活 | 收藏、分享 |

**交互细节**：
- 点击反馈：透明度0.8 + 轻微缩放0.98
- 加载状态：按钮内显示旋转loading图标
- 禁用状态：背景#ccc，无点击反馈

## 5. 动画与动效规范

### 5.1 页面转场

```
标准转场：滑动进入，200ms，ease-out
- 新页面从右侧滑入，透明度0→1
- 原页面向左偏移30%，透明度1→0.8

返回转场：滑动退出，200ms，ease-out
- 当前页面向右滑出
- 底层页面恢复原位
```

### 5.2 微交互动画

| 元素 | 触发 | 动画 | 时长 |
|-----|------|------|------|
| 按钮点击 | tap | scale(0.98) + opacity(0.9) | 100ms |
| 卡片选中 | tap | translateY(-2px) + shadow加深 | 200ms |
| 收藏心形 | tap | scale(1.2)→scale(1) + 填充 | 300ms |
| 数字变化 | 次数更新 | 数字滚动动画 | 500ms |
| 进度条 | 加载中 | 宽度增长 + 光效流动 | 持续 |
| 结果出现 | 页面加载 | 卡片依次淡入上滑，stagger 100ms | 500ms |

### 5.3 特殊动效

**八字排盘动画**：
- 四柱依次从下方滑入，每柱间隔200ms
- 天干地支分别旋转90度后定格
- 五行标签从中心扩散出现

**五行环形图**：
- 从0度开始绘制圆弧，1s完成
- 各区块依次高亮闪烁一次
- 最终定格在实际比例
